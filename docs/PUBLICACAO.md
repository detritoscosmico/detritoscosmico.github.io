# Publicação — Elegance Cosméticos

## Onde o site está publicado

Este site usa GitHub Pages no endereço:

https://detritoscosmico.github.io/

## Como testar localmente

1. Baixe ou clone o repositório.
2. Abra a pasta no VS Code.
3. Instale a extensão Live Server.
4. Clique com o botão direito em `index.html`.
5. Escolha `Open with Live Server`.

## Antes de publicar alterações

Checklist obrigatório:

- [ ] O site abre sem erro?
- [ ] O menu funciona no celular?
- [ ] Os botões de WhatsApp funcionam?
- [ ] As imagens carregam?
- [ ] Os preços estão corretos?
- [ ] O produto tem nome, categoria e descrição clara?

## Boas práticas

- Não subir imagens pesadas demais.
- Conferir links antes de divulgar.
- Separar campanha de produto novo em uma etapa própria.
- Testar no celular antes de postar nos grupos e Marketplace.
