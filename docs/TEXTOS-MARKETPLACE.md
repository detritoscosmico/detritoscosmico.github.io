# Textos prontos para Marketplace — Elegance Cosméticos

Use estes modelos para divulgar produtos no Facebook Marketplace, grupos locais, status do WhatsApp e mensagens diretas.

## Regras antes de postar

- Confirme estoque antes de anunciar.
- Atualize preço se necessário.
- Use foto real do produto quando possível.
- Não prometa resultado exagerado.
- Sempre leve o cliente para o WhatsApp.
