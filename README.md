# Elegance Cosméticos — Loja Virtual

Loja virtual estática da **Elegance Cosméticos**, publicada com GitHub Pages.

## Objetivo

Criar uma vitrine digital profissional para apresentar cosméticos, direcionar clientes para atendimento no WhatsApp e apoiar vendas pelo Instagram, Facebook, Marketplace e atendimento presencial.

## Stack atual

- HTML
- CSS
- JavaScript
- GitHub Pages

## Recursos atuais

- Home premium com banners promocionais
- Catálogo de cosméticos
- Categorias: Feminino, Masculino, Kits, Cabelos e Maquiagem
- Carrinho visual
- Compra direcionada para WhatsApp
- Suzy Beauty V2 com chat e recomendações por orçamento
